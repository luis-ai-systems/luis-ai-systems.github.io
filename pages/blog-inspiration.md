# 优秀个人博客与网站参考 (Reference Blog Inspiration)

为了持续优化本博客的视觉体验和功能，以下整理了一些在设计、技术栈、排版等方面表现出众的推拉、GitHub Pages 或独立博客，供后续作为借鉴、参考。

## 1. 极致设计与交互类 (Premium Design & Interactive)

| 网站名称 | 链接 | 亮点 |
| :--- | :--- | :--- |
| **Overreacted** | [https://overreacted.io/](https://overreacted.io/) | Dan Abramov 的博客。极其简洁的版式，完美的字体间距，明暗主题切换，专注内容。|
| **Vercel Blog** | [https://vercel.com/blog](https://vercel.com/blog) | 现代 Web 设计的风向标。极致的间距控制、Geist 字体、优雅的卡片阴影。 |
| **Lee Robinson** | [https://leerob.io/](https://leerob.io/) | 极简风格，出色的代码高亮方案，以及非常实用的内容筛选功能。 |
| **Paco Coursey** | [https://paco.me/](https://paco.me/) | 极致的苹果风（Apple-esque），丝滑的过渡动画，微交互非常到位。 |

## 2. 开发者与 GitHub 内容类 (Developer & Tech Focused)

| 网站名称 | 链接 | 亮点 |
| :--- | :--- | :--- |
| **GitHub Blog** | [https://github.blog/](https://github.blog/) | GitHub 官方博客。出色的多色调分类系统，以及非常有特色的配图设计。 |
| **Josh W. Comeau** | [https://www.joshwcomeau.com/](https://www.joshwcomeau.com/) | 互动式教学博客。充满趣味性的 UI 组件，文章中嵌入的可交互 Demo 非常棒。 |
| **Cassidy Williams** | [https://cassidoo.co/](https://cassidoo.co/) | 充满个性的风格，侧边栏与页脚设计非常有趣，整体氛围感很强。 |

## 4. 特色工具与资源 (Tools & Resources)

- **Mermaid.js**: 我们当前使用的架构图渲染方案，非常适合技术文档。
- **Giscus**: 基于 GitHub Discussions 的评论系统，比 Gitalk 更现代、更安全。
- **Inter Font**: 首页当前使用的字体，被认为是 UI 设计中最专业的无衬线字体。
